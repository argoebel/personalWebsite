PORTFOLIO

Haptik

Backend Development

September - December 2019

Haptik is a mobile application that helps medical professionals conduct resuscitation simulations for training healthcare workers and conducting medical research. This project is planned on being introduced to medical schools across North America.


This application was built on a Django backend with Postgres database with both React and React Native being used in the front end for the desktop and mobile viewing experiences respectively.


This was my first experience working on a real industry project and really opened my eyes to how software development works in the real world. Self-organizing teams, meeting milestones, client communication, and all aspects of Agile principles allowed us to complete this project on time and surpass client expectations.

Squawk

Backend Development

January - April 2020

Squawk is a decentralized, node-based social media platform (similar to diaspora*) in which users can store all personal data on privately-run servers allowing them to maintain data sovereignty while interacting with others online.


This was my second time using Django so I stepped right back into the backend development role and designed the APIs as well as Postgres database model. As this type of decentralized social media network required significant teamwork between groups, I collaborated heavily with other teams in the class to coordinate a universal API between groups that allowed us to share data freely between nodes.


This type of software project really pushed my software engineering and communication skills as it required maintaining internal data integrity, anticipating changes, and providing consistency to other nodes in the network.

HADA

Frontend Development

September - December 2018

HADA is an Android app designed around tracking medical ailments over time and seamlessly sharing information between patients and doctors.


This was my first time using Java to create an application. The Android Studio IDE was a great all-in-one tool that allowed us to integrate with Git, create interfaces, and test with the built-in emulator.


This was also my first time working in a team-based software engineering setting and it was a great learning experience. Working with Git, assigning roles, tracking issues, and team communication were the most important lessons learned with this project.
